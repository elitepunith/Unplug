# Unplug Reminder

A lightweight Windows app that replaces your standard shutdown button. It pops up a reminder to unplug your laptop charger before turning off your PC, helping preserve your battery's maximum capacity over time.

## Installation

1. Go to the Releases page and download UnplugReminder.exe.
2. Move the file to a permanent folder on your computer (like your Documents folder). 
3. Right-click the .exe file and select "Pin to taskbar".

## Usage

Instead of using the Windows Start menu to shut down, click the pinned app on your taskbar. It will show you a quick reminder to unplug your wire, and then safely execute the system shutdown.
